REST API (programaticky)

Jednoduché HTTP API pro vlastní skripty a integrace.

⚡
Agent API /api/agent/*
GET /api/agent/discovery Objevení schopností
GET /api/agent/status Stav kamery
POST /api/agent/ptz PTZ pohyb
POST /api/agent/ptz/stop Stop PTZ
POST /api/agent/analyze AI analýza scény
POST /api/agent/chat Chat s AI
POST /api/agent/speak TTS přes kameru
GET /api/agent/recordings Seznam záznamů
POST /api/agent/notify Poslat notifikaci
POST /api/agent/execute Generic execute
GET /api/agent/tools Seznam nástrojů 
curl -X POST http://YOUR_SERVER:8000/api/agent/ptz \
  -H "Content-Type: application/json" \
  -d '{"pan": 0.5, "tilt": -0.3}'
import httpx
r = httpx.post("http://YOUR_SERVER:8000/api/agent/execute",
    json={"action": "ptz", "params": {"pan": 0.5}})
print(r.json())

WebSocket (real-time stream)

Připojte se na WebSocket pro real-time video a ovládání.

📡
Live Stream WebSocket ws://SERVER:8000/ws/stream 
const ws = new WebSocket("ws://SERVER:8000/ws/stream");
ws.onmessage = (e) => {
  const data = JSON.parse(e.data);
  if (data.type === "frame") {
    img.src = "data:image/jpeg;base64," + data.data;
  }
  if (data.type === "motion_detected") {
    console.log("Pohyb!", data);
  }
};
ws.send(JSON.stringify({type:"ptz_move", pan:0.5, tilt:0}));
ws.send(JSON.stringify({type:"ptz_stop"}));
